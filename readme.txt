Display MobileMe gallery images on your blog.

== Description ==

This plugin displays your images in MobileMe Gallery page. Not only normal sidebar widgets, this plugin adds gallery at top of your first page first entry, too. There's setting panel at left side of admin page.

Requires:
 - WP 2.8 or later
 - PHP 5.x or later

*maybe this plugin will not run some server due to server permission issue. - reading rss file from outside of its domain and so on.

== Installation ==

Place the mobileme_gallery.php file in your /wp-content/plugins/ directory
and activate through the administration panel, and then go to the widget panel and
drag it to where you would like to have it!
